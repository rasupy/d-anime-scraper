d-anime-scraper リリースノート
=================================

最新バージョン: 0.2.x (2025-10-05 時点)

概要:
  d アニメストア「今期作品一覧」から作品情報を取得し、曜日ごとブロック CSV + 画像をフォルダに出力するツール。

--------------------------------------------------
[0.2.0] 2025-10-05
--------------------------------------------------
追加:
  - 曜日ごとブロック構造の CSV 出力 (曜日見出し/ヘッダ/空行区切り)。
  - 放送開始日を `10月6日～` の形式で表示 (API の schedule あるいは追加 JSON / 補完処理)。
  - 追加 JSON (new_tv_adddata_<season>.json) から放送開始日を補完する仕組み。
  - DEBUG=1 で raw.json 保存。UID/GID 可変実行 (.env: HOST_UID/HOST_GID)。
  - 画像ダウンロード機能 (タイトル名ファイル)。
変更:
  - 取得手段を静的 HTML パース → 公式 API (WS000118) + 追加 JSON 方式へ刷新。
  - Docker イメージを非 root ユーザー実行に変更。
修正:
  - 初期版で作品一覧が空 (JS 動的ロード未対応) だった問題。
既知の制約:
  - API から schedule が削除された作品は放送日が空欄のまま残る場合があります。
  - 早朝(0~3時)の翌日繰り上げソートは未完全 (JS 版ロジックの一部未移植)。
  - 詳細ページ HTML からの放送日抽出は現状フォールバックのみ (多くは追加 JSON で補完)。

--------------------------------------------------
[0.1.0] 2025-10-04
--------------------------------------------------
追加:
  - 初期実装 (暫定 HTML パース版)。
  - 単純 CSV (放送日(曜日),タイトル,画像URL) 出力。
  - Docker / docker-compose 実行。
